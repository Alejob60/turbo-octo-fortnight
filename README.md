# Orbital Prime Frontend

Landing page para Orbital Prime, una plataforma de infraestructura de inteligencia artificial para operaciones críticas.

## Tecnologías utilizadas

- Next.js 14
- React
- TypeScript
- Tailwind CSS
- Node.js

## Instalación

1. Clona el repositorio
2. Ejecuta `npm install` para instalar las dependencias
3. Ejecuta `npm run dev` para iniciar el servidor de desarrollo

## Características

- Diseño responsive
- Modo oscuro con efectos de "glow" azul eléctrico
- Integración con backend de MisYBot
- Componentes optimizados para experiencia de usuario empresarial

## Estructura del proyecto

```
src/
├── app/              # Rutas de la aplicación (App Router)
├── components/       # Componentes reutilizables
├── lib/             # Lógica de negocio y utilidades
└── styles/          # Archivos CSS globales
```
